# RiskProfiler Threat Intelligence assessment: "Handala"

This directory contains the pre-hire threat intelligence assessment deliverables on the threat actor **Handala** (Void Manticore / Storm-0842). 

## 📂 Project Structure

*   [**`index.html`**](file:///c:/Users/hp/Downloads/project%203/index.html): The interactive digital briefing and dashboard application.
*   [**`styles.css`**](file:///c:/Users/hp/Downloads/project%203/styles.css): Custom design tokens and styles (dark mode, glassmorphism, responsive grid, and print-media formatting).
*   [**`app.js`**](file:///c:/Users/hp/Downloads/project%203/app.js): Application logic (interactive ACH matrix calculator, workflow scrubber, SVG Diamond Model visualizer, and search/filters).
*   [**`intelligence_report.md`**](file:///c:/Users/hp/Downloads/project%203/intelligence_report.md): The formal, text-based threat intelligence report with TLP:AMBER markings on every section.

## 🚀 Key Features

1.  **Overview & Executive Summary**: A high-level briefing summarizing key judgments and introducing the authenticity calibration gauge (12% Grassroots Authenticity Verdict).
2.  **Attribution & ACH**: Analysis of Competing Hypotheses matrix evaluating whether Handala is a grassroots hacktivist crew, an Iranian state-sponsored front (MOIS), or a third-party false flag.
3.  **Operational Workflow**: An interactive, phase-by-phase walkthrough of the attack lifecycle, illustrating both traditional malware-centric and identity-centric cloud destruction vectors.
4.  **MITRE ATT&CK Mapping**: Observed techniques matched against technique IDs and specific, evidence-backed actor behaviors (searchable/filterable table).
5.  **Diamond Model of Intrusion**: An interactive SVG relationship chart representing Adversary, Capability, Infrastructure, and Victim connections.
6.  **Incident Case Studies**: Analysis of two high-profile claimed campaigns—the July 2024 CrowdStrike Phishing Wiper campaign and the March 2026 Stryker Corporation Microsoft Intune compromise.
7.  **IOC Database**: A database of defanged Indicators of Compromise (searchable table).
